# Project-3-AI-Recommendation-Logic
## Description
This project demonstrates a simple AI recommendation system using Python. It recommends an item based on the user's input.

## Features
- Simple recommendation logic
- User input based suggestions
- Easy to understand Python code
- Beginner-friendly project

## How It Works
1. Takes input from the user.
2. Processes the input using simple conditions.
3. Displays a suitable recommendation.

## Tools Used
- Python 3
- Pydroid 3
- GitHub

## Output
The program displays an AI recommendation based on the user's input.
